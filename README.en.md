## Overview

This repository collects course/practice notebooks for data understanding, NumPy, Pandas, visualization, preprocessing, linear regression, logistic regression, KNN, Titanic, iris, and housing-price prediction tasks.

Weekly folders include both notebooks and data files, making the repository a course-style path from data science basics to supervised-learning practice.

## Highlights

| Area | Description |
|---|---|
| Data analysis basics | Week 2 materials cover data understanding, NumPy, Pandas, and visualization. |
| Preprocessing | Week 3 materials cover structured and unstructured preprocessing. |
| Regression practice | Week 4 and assignment material cover linear regression and housing-price prediction. |
| Classification practice | Week 5 includes logistic regression, Titanic survival prediction, and iris classification. |
| Included datasets | CSV, Excel, text, and image data files are stored with the notebooks. |
